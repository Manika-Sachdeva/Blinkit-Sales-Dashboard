# Blinkit Sales Dashboard (Power BI Project)
## Project Objective
The Blinkit Sales Dashboard is designed to analyze retail and grocery sales performance by tracking total sales, item distribution, outlet performance, customer ratings, and product categories. The dashboard helps businesses understand sales patterns, optimize outlet operations, and improve decision-making using data-driven insights.

# Dataset Used
- <a href="https://github.com/Manika-Sachdeva/Blinkit-Sales-Dashboard/blob/main/BlinkIT%20Grocery%20Data.xlsx">Dataset</a>

## Questions (KPIS)
- What is the total sales revenue generated?
- What is the average sales value?
- How many items were sold?
- What is the average customer rating?
- Which outlet type generates maximum sales?
- Which outlet size contributes the highest revenue?
- Which outlet locations perform best?
- Which item types contribute the highest sales?
- How does fat content impact total sales?
- What are the outlet establishment trends over the years?

- Dashboard Interaction <a href="https://github.com/Manika-Sachdeva/Blinkit-Sales-Dashboard/blob/main/BLINKIT%20DASHBOARD.png">View Dashboard</a>

## Process
- Collected and cleaned grocery sales data using Power Query.
- Removed inconsistencies and handled missing values.
- Built data relationships and transformed datasets for analysis.
- Created DAX measures for:
  - Total Sales
  - Average Sales
  - Number of Items
  - Average Rating
- Designed interactive Power BI visualizations including donut charts, bar charts, KPI cards, and trend analysis graphs.
- Added slicers and filters for dynamic dashboard interaction.
